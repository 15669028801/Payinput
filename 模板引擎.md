## 二、ejs模板
* [中文文档](https://ejs.bootcss.com/)

### 简介
> "E" 代表 "effective"，即【高效】。EJS 是一套简单的模板语言，帮你利用普通的 JavaScript 代码生成 HTML 页面。EJS 没有如何组织内容的教条；也没有再造一套迭代和控制流语法；有的只是普通的 JavaScript 代码而已。

### 常用标签

> * <% %>流程控制标签
> * <%= %>输出标签（原文输出 HTML 标签）
> * <%- %>输出标签（HTML 会被浏览器解析）
